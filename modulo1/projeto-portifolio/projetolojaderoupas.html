Projeto Loja Venda de Roupas e Acessórios

Neste projeto desenvolvi um site de uma loja de vendas de Roupas e Acessórios, onde o o usuário poderia ter acesso a compras. Site único e exclusivo de vendas! Abaixo irei listar algumas das tecnologias utilizadas para a realização deste site.

Lista de Tecnologias Utilizadas.
  1. HTML 5
  2. Elementos de texto, de lista e de tabelas
  3. Elementos especiais
  4. Elementos de formulários
  5. Hierarquia de Aplicação
  6. Hierarquia de Seletores
  7. Algumas propriedades como: display, margin, padding
  8. Combinadores
  9. Seletores Especiais
  10. Centralização Horizontal
  11. Propriedade position: (static, relative, fixed, absolute e sticky)
