Projeto Loja de Cristais!

Neste projeto desenvolvi um site de uma loja de Cristais, onde o o usuário poderia ter acesso a todo o funcionamento da loja. Poderia verificar o estoque da loja, as vendas de determinado Cristal, valores e nome dos funcionários! Abaixo irei listar algumas das tecnologias utilizadas para a realização deste site.

Lista de Tecnologias Utilizadas.
  1. HTML 5
  2. Elementos de texto, de lista e de tabelas
  3. Elementos especiais
  4. Elementos de formulários
  5. Hierarquia de Aplicação
  6. Hierarquia de Seletores
  7. Algumas propriedades como: display, margin, padding
